# GlowAI – AI Powered Skin Analyzer & Cosmetic Recommendation System

## Project Structure
- `frontend/` — React + Vite + TypeScript frontend
- `backend/` — Flask REST API + AI pipeline

## Getting Started

### Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
